## 개요
``````
- 유형 : 개인 프로젝트
- 기간 : 2022.10.17 - 2022.11.20
- 소개 : 경기도 내 거주지 인근 동물 병원들의 위치와 보유 물품을 쉽게 찾을 수 있는 프로그램
- 주요 기술 : 
  1. Spring Security와 JWT를 통한 로그인 / 회원가입 구현
  2. Spring Batch와 Scheduling을 통해 경기도 내 동물 병원, 동물 약국 정보 저장
  3. Redis를 활용하여 동물 병원 조회 내용 캐싱하기
``````
